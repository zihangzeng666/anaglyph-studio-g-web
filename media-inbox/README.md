# 素材收件夾（media-inbox）

> **把新拍的截圖／照片／影片丟進這個資料夾就完成了。**
> 裁切、壓縮、轉格式、改進網站 —— 都由 Claude 處理。
> 你只要做對兩件事：**拍對畫面**、**用對檔名**。

---

## 命名規則（很重要）

- 檔名用 [SITE_BLUEPRINT.md](../SITE_BLUEPRINT.md) 素材總表裡的 **inbox 檔名**，一字不差
- 全部小寫、用 `-` 連字、不要中文、不要空格
- 同一格位拍了好幾張候選？加序號：`hero-shot-1.png`、`hero-shot-2.png`，Claude 幫你挑（或你在結構表備註欄指定）

---

## 拍攝規範

### 截圖（Studio G 介面）

- **全視窗**，深色主題，寬度 ≥1920（你螢幕的 2555×1352 全螢幕截圖完全可以，我來裁）
- **重點內容放在畫面上半部** —— 網站卡片是 16:9、裁切時以頂部對齊，太靠下會被切掉
- 畫面要「有東西」：表格有資料、數字有讀數、狀態是 done/live，不要空白頁
- 避免入鏡：Windows 工作列、時間、檔案總管、個人資料、報錯彈窗、`??` 亂碼

### 錄屏（影片）

- 1080p 以上、**10–25 秒**、mp4（其他格式也收，我轉）
- 錄之前先把狀態調好（例如 Track 已鎖定），錄的過程畫面穩定、不要急著晃
- 不用錄聲音 —— 網站上是靜音循環播放

### 實物照片（模具、貼紙、標定板、現場）

- **橫向構圖**、對焦清楚、光線均勻（避免一半亮一半暗）
- 背景乾淨：工作檯可以入鏡（有真實感），雜物、人手人腿盡量避開
- 手機拍即可，原圖直傳，不要先壓縮
